Project Overview

This project involved the comprehensive digital design of a production line for semi-automatic electric vehicle assembly. Working at original size specifications with CAD modeling, the project optimized efficiency while reducing overall production costs significantly.

Key Responsibilities

  • Production Line Design: Created full-scale CAD models of the assembly line with detailed robotic stations
  • Team Coordination: Led and coordinated 34 mechatronics students across multiple design phases
  • PLM Optimization: Streamlined Product Lifecycle Management phases to improve workflow efficiency
  • Budget Management: Reduced original budget from €490,000 while maintaining quality standards
  • Robot Programming: Configured and programmed ABB industrial robots using RobotStudio
  • Virtual Commissioning: Simulated production processes to validate design before physical implementation

Technical Implementation

The assembly line design integrates multiple robotic stations for efficient EV component assembly. Using advanced CAD tools and RobotStudio simulation, the design optimizes production flow while ensuring quality control. The project demonstrates how digital design and process optimization can significantly reduce production costs without compromising output quality.
