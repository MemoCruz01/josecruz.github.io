Project Overview

This project develops an autonomous system for harvesting radishes, combining a robotic arm with a computer vision system. The system identifies and locates ripe radishes using advanced cameras and vision algorithms, determining their exact position for UFactory's Lite6 robotic arm to pick them without damage. The robotic arm is mounted on an autonomous vehicle (AGV) chassis, while cameras and Raspberry Pi are installed in a protective case tilted at 45°, optimizing visual coverage and accuracy in agricultural automation.

System Architecture

  • Vision System: Dual-camera setup - 11.9MP RGB camera (Pi Camera Module 3 with Sony IMX708) plus Arducam ToF camera (0.43MP, 240×180) for distance measurement
  • Robotic Arm: UFactory Lite6 robotic arm mounted horizontally on AGV chassis front with optimized center of mass
  • Processing Unit: Raspberry Pi 5 (8GB) with real-time vision algorithms and kinematics processing
  • Field Configuration: Protective case mounted at 45° angle for maximum visual coverage and object tracking
  • AI Model: MobileNetV2 MultiHW neural network for radish detection with 70%+ accuracy even with blurred images

Technical Implementation

Vision Processing: The RGB camera identifies objects and colors using MediaPipe and OpenCV, while the ToF camera determines precise distance. Both cameras use the same 4:3 aspect ratio for accurate centroid projection. Image processing captures at 6 FPS with approximately 150ms inference time on Raspberry Pi 5.

AI Model: Trained using MediaPipe Model Maker with learning rate 0.45, batch size 8, and 75-90 epochs (avoiding overfitting). Models achieved 70%+ accuracy for radish detection including blurred or out-of-focus images. 16-bit quantization provides optimal balance between performance and accuracy.

Robotic Control: Kinematics calculations determine arm displacement using trigonometric functions. Coordinate conversion: Z-movement and X-movement calculated from object angle and distance. Arm velocity set to 30% to avoid chassis unbalance. Sequential Z-axis then X-axis movement prevents collisions and singularities.

System Integration: Multi-threaded Raspberry Pi enables simultaneous camera capture, AI inference, and arm control without delays. Communication system coordinates navigation state with arm permissive signals for synchronized harvesting during vehicle motion.

Development Process

1

System Design & Architecture

Defined requirements for autonomous radish harvesting. Designed mechanical integration with AGV, planned dual-camera vision system with 45° orientation for optimal coverage and accuracy.

2

Hardware Integration & Simulation

Integrated UFactory Lite6 arm on AGV chassis. Assembled Raspberry Pi 5 with RGB + ToF cameras. Created simulations in RoboDK for digital twin validation of harvesting process.

3

AI Model Development & Training

Trained MobileNetV2 MultiHW models for radish detection using MediaPipe Model Maker. Achieved 70%+ accuracy with 16-bit quantization. Optimized color detection HSV ranges (168-173, 148-209, 99-153).

4

Real-World Testing & Validation

Field tested autonomous harvesting system. Validated detection accuracy, arm coordination, and vehicle synchronization. Confirmed technical feasibility and efficiency of autonomous radish collection.
