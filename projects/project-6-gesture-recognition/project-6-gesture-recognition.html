Project Overview

This innovative project bridges human communication and robotics by creating a system that understands sign language gestures and translates them into precise robotic control commands. Conducted at Universität des Saarlandes in Germany, the project demonstrates the application of computer vision and machine learning in human-robot interaction.

Key Responsibilities

  • Computer Vision Development: Designed and implemented real-time hand detection and gesture recognition algorithms
  • Machine Learning Implementation: Trained and optimized ML models for accurate gesture classification
  • Gesture Analysis: Developed feature extraction techniques for hand pose and movement analysis
  • Robotic Integration: Created control interface between gesture recognition system and Braccio robotic arm
  • Writing Algorithm: Implemented kinematics solving and trajectory planning for arm writing
  • System Validation: Tested and refined system accuracy for reliable gesture interpretation

Technical Implementation

The system captures video input from a camera and processes it using computer vision techniques to detect hand positions and movements. Machine learning classifiers interpret these hand gestures as alphabetic characters or control commands. The recognized gestures are translated into coordinate data and passed to the Braccio robotic arm's control interface. The arm then moves according to trajectory planning algorithms to write the corresponding letters or words, demonstrating real-time human-robot communication.
