Project Overview

This project involved the complete design and implementation of an autonomous navigation system for mobile robots operating in real-world environments. Conducted in Bourges, France, the project focused on creating a robust ROS-based architecture capable of handling dynamic obstacles, sensor uncertainty, and real-time decision making.

Key Responsibilities

  • ROS Architecture: Designed and implemented complete ROS navigation stack for TurtleBot 4
  • SLAM Implementation: Developed Simultaneous Localization and Mapping algorithms for dynamic environments
  • Sensor Integration: Integrated multiple sensors including LiDAR, cameras, and IMU units
  • Path Planning: Implemented motion planning algorithms for obstacle avoidance and efficient trajectory generation
  • Real-World Testing: Conducted extensive testing in dynamic environments with validation and refinement
  • Performance Optimization: Optimized computational efficiency for real-time navigation processing

Technical Implementation

The navigation system integrates sensor fusion techniques to combine data from LiDAR, cameras, and inertial sensors. SLAM algorithms create dynamic maps of the environment while simultaneously determining robot position. Path planning modules use algorithms like RRT* and Dijkstra's for efficient navigation. The system handles real-time obstacle detection, dynamic obstacle avoidance, and continuous route optimization.
