Project Overview

The Autonomous Guided Vehicle (AGV) project focused on designing and manufacturing a fully autonomous indoor navigation system. This involved creating a complete platform capable of operating independently in structured environments with minimal human intervention.

Key Responsibilities

  • System Design & Manufacturing: Comprehensive design of the AGV platform including mechanical structure and electrical architecture
  • Team Leadership: Coordinated a multidisciplinary team of 11 engineers across mechanical, electrical, and software domains
  • ROS Navigation Pipeline: Implemented advanced autonomous navigation using Robot Operating System (ROS) with SLAM capabilities
  • Sensor Integration: Integrated multiple sensors (LiDAR, cameras, IMU) for environmental perception
  • Path Planning: Developed intelligent path planning algorithms for autonomous waypoint navigation
  • CAD/CAE/CAM Work: Executed complete digital design, simulation, and manufacturing workflow

Technical Implementation

The AGV platform utilizes cutting-edge robotics technologies to achieve autonomous navigation in indoor environments. The ROS-based architecture enables real-time sensor fusion, dynamic path planning, and obstacle avoidance. Comprehensive testing in operational environments validated the system's robustness and reliability.
