# 📚 Documentation Folder

This folder contains all the guides and documentation for your portfolio template.

## 📁 Files Inside

### **PORTFOLIO_ARCHITECTURE.mmd**
- **Format:** Mermaid diagram file (pure diagram code)
- **Best for:** Quick visual reference of how the template is structured
- **View with:** Markdown Preview Mermaid Support extension
- **Shows:** File relationships, dependencies, and component structure

### **PORTFOLIO_ARCHITECTURE.md**
- **Format:** Markdown file with embedded diagram + explanations
- **Best for:** Understanding both the visual structure AND how things work
- **View with:** Any markdown viewer or VS Code preview
- **Shows:** Diagram + detailed explanations of each component

### **TEMPLATE_GUIDE.md**
- **Location:** Root folder (you can move it here if you want)
- **Format:** Comprehensive markdown guide
- **Best for:** Learning how to make modifications
- **Contains:** Step-by-step tutorials, code examples, tips

---

## 🎯 Quick Guide to Files

| File | Purpose | When to Use |
|------|---------|------------|
| `.mmd` file | Visual diagram only | Need quick visual reference |
| `.md` file | Diagram + explanations | Want to understand the system |
| `TEMPLATE_GUIDE.md` | How-to guide | Ready to make changes |

---

## 💡 Tips

✅ Open `.md` files in VS Code and press **Ctrl+K, V** to see preview  
✅ Open `.mmd` files with your new extension for visual rendering  
✅ Print or export diagrams as images if needed  
✅ Keep this folder organized as you add more guides
